# 🎮 AI Gaming Performance Analyzer

An AI-based system that analyzes gameplay videos to evaluate player performance using **YOLOv8 and OpenCV**.

---

## 🚀 Features

* 🎯 Enemy detection using YOLOv8
* 🔫 Shot detection using frame differencing
* ⏱️ Reaction time calculation
* 🎮 Movement analysis
* 📊 Automatic performance report generation

---

## 🧠 Tech Stack

* Python
* OpenCV
* YOLOv8 (Ultralytics)

---

## 📊 Sample Output

| Metric              | Value   |
| ------------------- | ------- |
| Avg Reaction Time   | 622 ms  |
| Best Reaction Time  | 66 ms   |
| Worst Reaction Time | 1999 ms |

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```
